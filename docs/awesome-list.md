# Awesome List

A list of projects using or extending `Exponax`:

Feel free to submit a PR if you have a project that uses `Exponax` and you want to share it here!

**APEBench**: [GitHub](https://github.com/tum-pbs/apebench)  
A benchmark suite for autoregressive neural emulation of PDEs, using `Exponax`
for fast procedural data generation and for solver-in-the-loop neural-hybrid
models.

**Exponax-SPDE**: [GitHub](https://github.com/GVourvachakis/exponax-spde)  
Stochastic Allen-Cahn steppers.
